===============================================
    FILE RENAMER PRO - Quick Start Guide
===============================================

SETUP
-----
1. Copy this entire folder to your USB drive or computer
2. Double-click "FileRenamerPro.exe" to run

NO INSTALLATION REQUIRED - runs directly from USB!


HOW TO USE
----------
1. Click "Browse..." to select your Job Folder
   - Job number auto-extracts from folder name

2. CLOCK IN to start tracking your time

3. Select your options:
   - Product SKU (from dropdown)
   - File Purpose (PROOF, PRINT, WEB, etc.)
   - Artwork Reference (type a brief description)
   - Revision auto-detects from existing files!

4. Drag & drop files OR click "Add Files..."

5. Click "RENAME FILES"
   - Files are renamed and moved to the job folder

6. CLOCK OUT when done - time is logged!


FILE FORMAT
-----------
Files are renamed to:
   <Job#>_<SKU>_(<ArtRef>)_<Purpose>_<Rev>.<ext>

Example:
   12345_MUG-11OZ_(BlueDog)_PROOF_1.psd


CUSTOMIZATION
-------------
Edit "config.json" to customize:
- Product SKU list
- File purposes
- Revision options
- Timer warning time
- Job folder base directory


TIME LOGS
---------
Time logs are saved in a "time_logs" folder
(created automatically next to the .exe)

Format: timelog_YYYY-MM-DD.json


TIPS
----
- The app remembers the job folder during your session
- Revision auto-increments after each rename
- Timer flashes orange after 30 minutes as a reminder
- Config changes take effect after restarting the app


SUPPORT
-------
Contact your system administrator for help.
